 SafeTap

SafeTap is a UI/UX prototype for an emergency safety app focused on fast SOS activation, live tracking, and nearby responder support.

## Overview

This project currently contains a single interactive HTML prototype that demonstrates the SafeTap emergency experience from onboarding to active alert response.

Core ideas shown in the prototype:

- emergency SOS activation
- nearby "Keeper" responder network
- live tracking during active alerts
- emergency contacts and permissions setup
- safety-focused mobile onboarding flow

## Project File

- `keepersos_complete_ui_ux.html` - the main interactive prototype

## How To Run

1. Open `keepersos_complete_ui_ux.html` in your browser.
2. Use the on-screen navigation buttons to move between screens.
3. Tap the SOS button on the Home screen to view the active alert flow.

## Prototype Screens

## Design Notes

The prototype uses:

- HTML for structure
- CSS for styling and animations
- inline JavaScript for screen navigation and interactions
- Google Fonts for typography

## Purpose

SafeTap is intended as a concept prototype for presenting the user experience of a personal safety and rapid-response mobile app.
